# BharatView Business Summit & Awards 2026

## Overview
A premier leadership platform website designed to showcase the BharatView Business Summit & Awards 2026. Built with React and TailwindCSS.

## Project Structure
- `index.html`: Entry point with Tailwind configuration and theme variables.
- `app.js`: Main React application wrapper.
- `components/`: Modular UI components (Header, Hero, Overview, Highlights, WhoShouldAttend, Awards, ComingSoon, Footer).
